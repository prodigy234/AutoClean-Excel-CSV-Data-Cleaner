# 📊 Excel & CSV Data Cleaner  

## 🚀 Overview  
**Excel & CSV Data Cleaner** is a powerful **Streamlit** web app that simplifies data preprocessing by automating the cleaning of Excel and CSV files. Upload your dataset, select transformations, and get a clean, standardized file in seconds!  

## ✨ Features  
✅ **Column Name Formatting** – Convert to `snake_case` or `camelCase`  
✅ **Text Transformations** – Convert to **Title Case, UPPER, lower**  
✅ **Missing Data Handling** – Fill NaN values using **Mean, Median**  
✅ **Date Standardization** – Convert date formats automatically  
✅ **Data Deduplication** – Remove duplicate records effortlessly  
✅ **Special Character Removal** – Clean unwanted symbols from text  
✅ **Instant Download** – Get your cleaned dataset in one click  

## 📦 Installation  
To run this app locally, install the required dependencies:  
```bash
pip install streamlit pandas openpyxl xlrd
```

## ▶️ Usage  
Run the app with:  
```bash
streamlit run app.py
```
Then, upload your **Excel (.xls, .xlsx, .csv)** file, apply transformations, and download your cleaned data.
